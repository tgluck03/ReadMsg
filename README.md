# **Outlook to PDF Converter Tool**

This tool allows you to easily convert `.msg` and `.eml` files from Outlook to PDF format with a simple drag-and-drop interface. Additionally, it automatically saves the attachments of the email in the same folder for easy access.

## **Features**

- **Drag-and-drop functionality** for `.msg` and `.eml` files.
- **Convert Outlook emails** to PDF format.
- **Save email attachments** in the same folder as the converted PDF.
- Simple, easy-to-use interface with minimal setup.

## **How to Use**

1. **Download and Install the Tool:**
   - Download the latest release from the repository (or setup instructions if required).
   - No installation needed! Just run the executable.

2. **Convert .msg or .eml Files:**
   - Open the tool.
   - Simply drag and drop your `.msg` or `.eml` files onto the tool's window.
   - The email will be converted into a PDF format.
   - All attachments will be saved in the same folder as the converted PDF.

3. **Find Your Converted PDFs:**
   - The output PDF will have the same name as the email file.
   - All attachments (if present) will be saved in the same folder as the converted PDF.

## **Supported File Types**

- `.msg` (Outlook message files)
- `.eml` (Email message files)
